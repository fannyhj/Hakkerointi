# ISMS-soveltamisala - Scope

Sisällytän scopeeni kotitalouteni IT-ympäristön, jonka laitteita käytän kurssitehtävissäni sekä opiskelussani. Soveltamisala sisältää kotiverkkoni perusinfrastruktuutin, laitteet ja niissä käsiteltävät tiedot.

Kotiverkkoni perusinfrastruktuuriin kuuluu reititin, sen palomuuritoiminnot ja Wi-Fi verkko. Kurssiharjoituksiin käytettävät laitteet ovat oma kannettava- sekä pöytätietokone, virtuaalikone (Linux VM) sekä älypuhelin.

Tiedot joita kuuluu suojella ovat kurssimateriaalit, harjoitusraportit, GitHub sivustoni sekä kurssiin liittyvät tunnukset ja salasanat.
Soveltamisalan ulkopuolelle rajaan äly-tv:n, sillä tämä ei ole merkityksellinen laite kurssitehtävien tietoturvan kannalta.


Keskeiset rajapinnat ovat; 

Internet-yhteys: reititin ja palomuuri

Pilvipalvelut: GitHub ja OneDrive

Etäyhteydet: SSH

Toimittajat: Inernet-operaattori, laitevalmistajat ja pilvipalveluntarjoajat

 ## Evidence-lisä:

Virtuaalikoneet: VM-listaus tai hypervisorin näkymä
